# 🌦️ Weather App

A responsive Weather App built using HTML, CSS, and JavaScript that allows users to check real-time weather information for any city.

## 🚀 Features

* Search weather by city name
* Displays temperature, weather condition, humidity, and wind speed
* Responsive design for mobile and desktop devices
* User-friendly interface
* Real-time weather data using API

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API
